Approved By: Heidi
Button is an extension to standard button element with icons and theming. Button is either applies as a component using p-button element or a directive using pButton attribute. Directive enhances an existing button whereas p-button is an element on its own. 
                                        
                                                <button pButton type="button" label="Primary" ></button>
                                                <p-button label="Click" ></p-button>
                                        
                                    
                                        
                                            <button pButton type="button" label="Secondary" class="p-button-secondary">
                                        
                                    
                                    
                                        <p-button label="Disabled" disabled="true">
